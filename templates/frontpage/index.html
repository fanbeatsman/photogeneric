Antmap

Webapp that parses data strings and aggregates the data into a SQL database. Django backend ensures reliable MVC control flow and leverages the backend data to coordinate a frontend visualization including map location and dynamically ordered tables.

Photogeneric

Lightweight photo gallery app done for a friend photographer. It features 2 visualization methods "S" (linear) and "H" (history mode). Again, it sits atop a Django backend, but with the bulk of its logic in the frontend held together with an array of libraries such as JQuery, Packery, Skrollr and Bootstrap.

Pokercast

The fruit of my first hackathon and first time working in collaboration with other people. Pokercast is a poker game built on the Chromecast that allows up to 6 players to play poker. The board is displayed on a TV and each player's hand is displayed on their smartphone. It includes a RESTful backend server, an iOS app and a frontend composed of a browser based UI and Chromecast API.
